# java-desktop-buildr-swing

## Description
This is a POC project to demonstrate a
java swing application running inside a
docker container. 

## Tech stack
- buildr
- java
  - swing

## Docker stack
- vanto/apache-buildr:latest-jruby-jdk8
- alpine:edge

## To run
`sudo ./install.sh -u`

## To see help
`sudo ./install.sh -h`
